# DesafioAPI

Criar uma aplicação spring-boot para cadastro de incidentes. A aplicação deverá fornecer operações
REST que possibilitem o cadastro/manutenção/remoção de incidentes.
Um incidente é composto pelos campos:
Incident
	- idIncident
	- name
	- description
	- createdAt
	- updatedAt
	- closedAt
